# Card-Game-CSharp.NET
This is a simple card-based game built using C# and Windows Forms in Visual Studio.
Created this school porject that demonstrates the use of object-oriented programming, event handling, and UI design using the .NET Framework.

This project was created for educational and personal learning purposes.

# Features
1. Basic gameplay logic using C#.
2. Graphical user interface built with Windows Forms.
3. Simple layout for easy modification and experimentation.

# Requirements
- Visual Studio 2019 / 2022 (Community Edition or higher)
- .NET Framework 4.x (or the version your project targets)
- Windows OS (since this uses WinForms)

# How to Run the Project
1. Clone or download this repository.
2. Open the folder in Visual Studio.
3. Locate the solution file (.sln).
4. Double-click to open it in Visual Studio.
5. In the Solution Explorer, open the Form1.cs file and click Run or press F5.
